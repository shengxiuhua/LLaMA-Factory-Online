# 2.3 新用户限时免费体验

活动期间，LLaMA-Factory Online为每位新用户提供限时免费体验权益。新用户注册后，平台将自动发放**新用户注册有礼**代金券；扫描下方二维码进入用户交流群，还可再领取一张**新用户进群金喜**代金券。在使用相关服务时，将按照平台计费规则进行抵扣。

![控制台](https://s1.llamafactory.online/llamafactory-online/docs/v2.0/documents/quickStart/getstarted/%E6%89%AB%E7%A0%81%E5%85%A5%E7%BE%A4.png)



## 2.3.1 试用须知

* 一次机会：面向LLaMA-Factory Online平台新用户发放一定面额的活动代金券，每个账户仅可获取一次。

* 注册/登录后使用：活动代金券在新用户注册之后发放。一旦使用过产品，将无法再获取新用户专属的活动代金券。

* 额度通用：发放的活动代金券额度通用，无使用门槛，可用于模型微调、模型预训练、模型评估、数据处理、Chat。

* 到期停止：活动代金券有效期30天，有效期剩余7天系统自动发送提醒，若账户余额不足且未及时充值时任务将被中止。

## 2.3.2 开始试用

1. 用户访问[LLaMA-Factory Online](https://www.llamafactory.online)官网，找到右上角“登录”按钮。根据页面提示完成注册/登录。
2. 点击左侧导航栏“实例空间”，在操作区单击“开始微调”按钮，进入资源配置页面，选择镜像（下图高亮①）；选择资源规格（下图高亮②），配置适用于模型训练的资源，确定卡数；选择价格优惠方式（下图高亮③），单击“启动”按钮（下图高亮④），创建实例。

![控制台](https://s1.llamafactory.online/llamafactory-online/docs/v2.0/documents/quickStart/23.png)
         
   

## 查询资源使用状态

当前平台[实例模式]下仅支持单一实例的运行。用户成功登录后，可通过选择“控制台”菜单项进入操作界面。若页面正确显示了LLaMA-Factory快速微调模型、VSCode处理专属数据和JupyterLab处理专属数据三个部分，则表明实例已成功启动并处于正常运行状态。同时，系统根据现有余额以及资源使用单价，自动估算可用时长，如下图所示。

![查询使用状态](https://s1.llamafactory.online/llamafactory-online/docs/v2.0/documents/quickStart/03.png)

> **重要**
> 
> LLaMA-Factory Online计费存在延迟。您在停止服务后收到的扣费通知，是对停止前已使用资源的结算，并非额外收费。为避免账户欠费，建议在资源耗尽前及时停止实例。
